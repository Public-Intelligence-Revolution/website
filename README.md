# Public Intelligence Website

The official website for **Public Intelligence** — a globally distributed, community-owned AI infrastructure.

The website communicates the project's vision, architecture, research, roadmap, and development progress.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS

## Pages

- Home
- Vision
- Architecture
- Research
- Roadmap
- Contribute

## Development

```bash
npm install
npm run dev
```

Visit:

```
http://localhost:3000
```
